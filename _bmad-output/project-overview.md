## Purpose

**project-rulebook-be** is an AI-powered board game rules generation backend that:

1. **Identifies board games** from uploaded box photos using AI vision
2. **Generates comprehensive rule summaries** for identified games
3. **Manages user accounts** with authentication and profiles
4. **Optimizes AI costs** through intelligent caching (80% reduction)
5. **Supports pre-launch signups** via email waitlist
